# 🧪 Historia de la Química - Proyecto Interactivo

Este proyecto es una página web educativa con:
- 📖 Información básica sobre la historia de la química
- ❓ Juego de preguntas (quiz)
- 🧠 Juego de memoria

## 🚀 Cómo usar

1. Descarga o clona el repositorio
2. Abre `index.html` en tu navegador

## 🛠 Tecnologías
- HTML
- CSS
- JavaScript

## 🎯 Objetivo
Aprender química de forma divertida e interactiva.
